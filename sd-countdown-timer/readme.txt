A simple countdown timer plugin that allows you to display a countdown timer using a shortcode.

== Description ==

The Countdown Timer plugin allows you to display a countdown timer on your WordPress site. You can easily set the target date and time via a shortcode. The timer counts down to the specified date and time, and displays days, hours, minutes, and seconds.

== Installation ==

1. Upload the `countdown-timer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[sd_countdown_timer date="YYYY-MM-DD HH:MM:SS"]` shortcode to display the countdown timer.

== Frequently Asked Questions ==

= How do I use the countdown timer? =

Simply add the shortcode `[sd_countdown_timer date="YYYY-MM-DD HH:MM:SS"]` to any post, page, or widget where you want the countdown timer to appear.

= Can I use multiple countdown timers on a single page? =

Yes, you can use the shortcode multiple times on a single page. Each timer will function independently.

= What happens when the countdown ends? =

When the countdown ends, a message will be displayed indicating that the countdown has ended.
